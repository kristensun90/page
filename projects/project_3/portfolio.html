This animation explores the surreal transformation of a kitten into a human, blending mystery and dreamlike visuals. Created using charcoal, the piece embraces a raw, textured aesthetic. Panda Puzzle

This interactive piece is a jigsaw puzzle featuring pandas. Designed for collaborative engagement, it invites friends to assemble different sections together. When completed, the playful fragments unite into a single, cohesive image—highlighting the joy of shared creation.
